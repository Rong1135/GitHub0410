如何建立 Git Local Repository(本端數據庫)

新建數據庫

  1. 首先建立一個新專案的資料夾
  2. 開啟Git Bash，並使用 cd 前往資料夾
  3. 執行 git init 指令建立本地數據庫
    • 若執行成功，資料夾內會出現一個隱藏的 .git 資料夾
  4. 在新專案資料夾內新增 index.html 檔案
  5. 使用 git add，將檔案加入索引
  6. 使用 commit -m "填寫版本資訊" 提交新版本

Git status 狀態

  1. Untracked file 代表偵測到有新增檔案，但 Git 並未追蹤
  2. Changes to be committed 表示檔案有成功被加到索引
  3. Modified: 檔案有被追蹤

Git 基礎指令

  • 單一檔案加入索引: git add <檔案名稱>
  • 所有檔案加入索引: git add .
  • 查看當前狀態: git status
  • 提交版本: git commit -m "填寫版本資訊"
  • 提交的歷史紀錄: git log

從 working directory(工作目錄) add 到 staging area(索引) commit 到 Local Repository(本地數據庫)
