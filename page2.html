Git basic operations

如何建立 Git Local Repository(本端數據庫)

新建數據庫

  1. 首先建立一個新專案的資料夾
  2. 開啟Git Bash，並使用 cd 前往資料夾
  3. 執行 git init 指令建立本地數據庫
    1. 若執行成功，資料夾內會出現一個隱藏的 .git 資料夾

版本控制流程

  1. 使用 git init，成功建立數據庫
  2. 新增一個 index.html 檔案
  3. 使用 git status 指令查看狀態
    1. Untracked file 代表偵測到有新增檔案，但 Git 並未追蹤
    2. Changes to be committed 表示檔案有成功被加到索引
  4. 將檔案加入到索引(staging area): 指令 git add <檔案名稱>
  5. 一次全加入則使用 git add .
  6. 提交新版本: git commit -m "<填寫版本資訊>"
  7. git log : 查詢版本
  1. 開新資料夾，git init 建立數據庫
  2. 新增一個 index.html 檔案
    • 單一檔案加入索引: git add 檔案名稱(檔案名稱有加大於小於)
    • 所有檔案加入索引: git add .
  4. 提交版本: git commit -m "新增網頁"
  5. 觀看歷史紀錄: git log
